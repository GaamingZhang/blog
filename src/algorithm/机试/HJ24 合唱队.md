# HJ24 合唱队

## 描述

音乐课上，老师将 $n$ 位同学排成一排。老师希望在不改变同学相对位置的前提下，从队伍中选出最少数量的同学，使得剩下的同学排成合唱队形。

记合唱队形中一共有 $k$ 位同学，记编号为 $1, 2, \dots, k$，第 $i$ 个人的身高为 $h_i$。要求：存在一位同学编号为 $i$ ($1 < i < k$)，使得 $h_1, h_2, \dots, h_{i-1}$ 严格递增，且 $h_{i+1}, h_{i+2}, \dots, h_k$ 严格递减；更具体地，合唱队形呈 $h_1 < h_2 < \cdots < h_{i-1} < h_i$、$h_i > h_{i+1} > \cdots > h_k$。

你能帮助老师计算，最少需要出列多少位同学，才能使得剩下的同学排成合唱队形？

## 输入描述

第一行输入一个整数 $n$ ($1 \le n \le 3000$) 代表同学数量。

第二行输入 $n$ 个整数 $h_1, h_2, \dots, h_n$ ($0 \le h_i \le 10^5$) 代表每一位同学的身高。

## 输出描述

输出一个整数，代表最少需要出列的同学数量。

## 示例1

输入：

```
8
186 186 150 200 160 130 197 200
```

输出：

```
4
```

说明：

在这个样例中，有且仅有两种出列方式，剩下的同学分别为 $\{186, 200, 160, 130\}$ 和 $\{150, 200, 160, 130\}$。

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int cnt;
    vector<int> nums;
    cin >> cnt;
    for (int i = 0; i < cnt; ++i) {
        int tmp;
        cin >> tmp;
        nums.push_back(tmp);
    }

    if (nums.size() == 0) return 0;
    vector<int> dp1(cnt, 1), dp2(cnt, 1);
    for (int i = 0; i < cnt; ++i) {
        for (int j = 0; j < i; ++j) {
            if (nums[i] > nums[j]) {
                dp1[i] = max(dp1[i], dp1[j] + 1);
            }
        }
    }

    for (int i = cnt - 1; i >= 0; --i) {
        for (int j = cnt - 1; j > i; --j) {
            if (nums[i] > nums[j]) {
                dp2[i] = max(dp2[i], dp2[j] + 1);
            }
        }
    }

    int maxLength = 0;
    for (int i = 0; i < cnt; ++i) {
        if (maxLength < dp1[i] + dp2[i] - 1) {
            maxLength = dp1[i] + dp2[i] - 1;
        }
    }
    cout << cnt - maxLength << endl;
    return 0;
}
```
