# HJ178 【模板】双指针

对于给定的长度为 $n$ 的数组 $\{a_1, a_2, \ldots, a_n\}$，找出最长的区间，满足区间中元素两两不同。如果有多个这样的区间，依次输出它们。

第一行输入一个整数 $n$ ($1 \le n \le 2 \times 10^5$) 代表数组中的元素数量。第二行输入 $n$ 个整数 $a_1, a_2, \ldots, a_n$ ($0 \le a_i \le n$) 代表初始数组。

第一行输出一个整数 $m$ ($1 \le m \le n$) 代表满足条件的区间数量。此后 $m$ 行，每行输出两个整数 $l, r$ ($1 \le l \le r \le n$) 代表满足条件的区间。本题没有 SPJ，请按照 $l$ 递增的顺序输出。

示例1

输入：

```
6
1 1 4 5 1 4
```

输出：

```
3
2 4
3 5
4 6
```

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> nums;
    for (int i = 0; i < n; ++i) {
        int tmp;
        cin >> tmp;
        nums.push_back(tmp);
    }

    vector<int> last(n + 1, -1);
    int left = 0, right = 0, maxLen = 0;
    vector<pair<int, int>> res;

    while (right < n) {
        if (last[nums[right]] >= left) {
            left = last[nums[right]] + 1;
        }
        last[nums[right]] = right;
        int curLen = right - left + 1;
        if (curLen > maxLen) {
            maxLen = curLen;
            res.clear();
            res.push_back({left + 1, right + 1});
        } else if (curLen == maxLen) {
            res.push_back({left + 1, right + 1});
        }
        ++right;
    }

    cout << res.size() << endl;
    for (int i = 0; i < res.size(); ++i) {
        cout << res[i].first << " " << res[i].second << endl;
    }
    return 0;
}
```
