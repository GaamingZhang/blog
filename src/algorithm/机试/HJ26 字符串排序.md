# HJ26 字符串排序

## 描述

对于给定的由可见字符和空格组成的字符串，按照下方的规则进行排序：

- 按照字母表中的顺序排序（不区分大小写）
- 同一字母的大小写同时存在时，按照输入顺序排列
- 非字母字符保持原来的位置不参与排序

直接输出排序后的字符串。

字符串由 ASCII 码在 $32$ 到 $126$ 范围内的字符组成。

## 输入描述

在一行上输入一个长度为 $1 \le length(s) \le 1000$，由可见字符组成的字符串 $s$。

## 输出描述

输出一个字符串，代表按照规则排序后的字符串。

## 示例

### 示例1

输入：

```
BabA
```

输出：

```
aABb
```

### 示例2

输入：

```
Hello NowCoder!
```

输出：

```
CdeeH llNooorw!
```

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string str;
    getline(cin, str);
    vector<char> letter;
    
    for (int i = 0; i < 26; ++i) {
        for (int j = 0; j < str.size(); ++j) {
            if (str[j] - 'a' == i || str[j] - 'A' == i) {
                letter.push_back(str[j]);
            }
        }
    }
    
    for (int i = 0, k = 0; i < str.size() && k < letter.size(); ++i) {
        if ((str[i] >= 'a' && str[i] <= 'z') || (str[i] >= 'A' && str[i] <= 'Z')) {
            str[i] = letter[k++];
        }
    }
    
    cout << str << endl;
    return 0;
}
```

## 解题思路

1. **提取字母字符**：遍历字符串，按照 a-z 的顺序提取所有字母字符，保持原始的相对顺序
2. **稳定排序**：由于按照字母表顺序遍历，同一字母的大小写会按照输入顺序排列
3. **替换原字符串**：再次遍历原字符串，将字母位置替换为排序后的字母，非字母字符保持不变

这种方法的时间复杂度为 $O(26 \times n) = O(n)$，空间复杂度为 $O(n)$。
