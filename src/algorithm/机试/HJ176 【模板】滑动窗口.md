# HJ176 【模板】滑动窗口

给定一个长度为 $n$ 的整数数组 $a$ 和一个窗口大小 $k$ ($1 \le k \le n$)。滑动窗口从左到右移动，每次右移一位（窗口覆盖下标 $[i, i+k-1]$）。对于数组的每一个窗口位置，求出窗口内元素的最大值。

第一行输入两个整数 $n, k$ ($1 \le k \le n \le 2 \times 10^5$)。第二行输入 $n$ 个整数 $a_1, a_2, \ldots, a_n$，元素范围 $1 \le a_i \le 10^9$。

输出共 $n-k+1$ 个整数，为每个滑动窗口的最大值，数之间以单个空格分隔。

示例1

输入：

```
10 3
2 13 6 19 15 13 17 9 19 13
```

输出：

```
13 19 19 19 17 17 19 19
```

示例2

输入：

```
10 1
13 13 5 3 9 19 18 4 17 3
```

输出：

```
13 13 5 3 9 19 18 4 17 3
```

示例3

输入：

```
10 10
15 20 5 20 19 1 4 18 14 15
```

输出：

```
20
```

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    vector<int> nums, res;
    for (int i = 0; i < n; ++i) {
        int tmp;
        cin >> tmp;
        nums.push_back(tmp);
    }

    deque<int> q;
    for (int i = 0; i < n; ++i) {
        while (!q.empty() && nums[i] >= nums[q.back()]) {
            q.pop_back();
        }
        q.push_back(i);
        while (q.front() <= i - k) {
            q.pop_front();
        }
        if (i >= k - 1) {
            res.push_back(nums[q.front()]);
        }
    }
    for (int i = 0; i < res.size(); ++i) {
        if (i > 0) {
            cout << " ";
        }
        cout << res[i];
    }
    cout << endl;
    return 0;
}
```
