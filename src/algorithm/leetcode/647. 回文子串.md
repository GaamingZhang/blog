# 647. 回文子串

给你一个字符串 s ，请你统计并返回这个字符串中 回文子串 的数目。

回文字符串 是正着读和倒过来读一样的字符串。

子字符串 是字符串中的由连续字符组成的一个序列。

示例 1：
输入：s = "abc"
输出：3
解释：三个回文子串: "a", "b", "c"

示例 2：
输入：s = "aaa"
输出：6
解释：6个回文子串: "a", "a", "a", "aa", "aa", "aaa"

```cpp
class Solution {
public:
    int countSubstrings(string s) {
        int cnt = 0;
        for (int i = 0; i < s.size(); ++i) {
            int start = i, end = i;
            while (start >= 0 && end < s.size() && s[start] == s[end]) {
                --start;
                ++end;
                ++cnt;
            }

            start = i, end = i + 1;
            while (start >= 0 && end < s.size() && s[start] == s[end]) {
                --start;
                ++end;
                ++cnt;
            }
        }
        return cnt;
    }
};
```
