# 61. 旋转链表

给你一个链表的头节点 head ，旋转链表，将链表每个节点向右移动 k 个位置。

示例 1：
输入：head = [1,2,3,4,5], k = 2
输出：[4,5,1,2,3]

示例 2：
输入：head = [0,1,2], k = 4
输出：[2,0,1]

```cpp
class Solution {
public:
    ListNode* rotateRight(ListNode* head, int k) {
        if (head == nullptr) {
            return head;
        }

        ListNode* iter = head;
        int count = 1;
        while (iter->next != nullptr) {
            iter = iter->next;
            ++count;
        }
        k = k % count;
        if (k == 0) {
            return head;
        }

        int prevCount = count - k;
        ListNode* prevIter = head;
        while (--prevCount > 0) {
            prevIter = prevIter->next;
        }
        ListNode* result = prevIter->next;
        prevIter->next = nullptr;
        iter->next = head;
        return result;
    }
};
```
