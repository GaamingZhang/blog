# 309. 买卖股票的最佳时机含冷冻期

## 一、题目理解

给定一个整数数组prices，其中第  prices[i] 表示第 i 天的股票价格 。​

设计一个算法计算出最大利润。在满足以下约束条件下，你可以尽可能地完成更多的交易（多次买卖一支股票）:

- 卖出股票后，你无法在第二天买入股票 (即冷冻期为 1 天)。
- 注意：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

### 示例 1:
输入: prices = [1,2,3,0,2]
输出: 3 
解释: 对应的交易状态为: [买入, 卖出, 冷冻期, 买入, 卖出]

### 示例 2:
输入: prices = [1]
输出: 0

## 二、解法思路

使用**记忆化搜索**（DFS + 缓存）的方法，定义状态函数：
- **`dfs(i, hold)`**：表示第 `i` 天，持有状态为 `hold` 时的最大利润
  - `i`：当前天数（0-based）
  - `hold`：持有状态（0 表示不持有，1 表示持有）

### 状态转移方程

**当持有股票时**（`hold = 1`）：
```
dfs(i, 1) = max(
    dfs(i-1, 1),                // 继续持有（前一天也持有）
    dfs(i-2, 0) - prices[i]     // 买入（必须前一天是冷冻期，即 i-2 天不持有）
)
```

**当不持有股票时**（`hold = 0`）：
```
dfs(i, 0) = max(
    dfs(i-1, 0),                // 继续不持有
    dfs(i-1, 1) + prices[i]     // 卖出
)
```

### 边界条件

- **当 `i < 0` 时**：
  - 如果 `hold = 1`（持有）：返回 `INT_MIN`（不可能的状态）
  - 如果 `hold = 0`（不持有）：返回 `0`（初始状态利润为 0）

### 记忆化缓存

使用二维数组 `memo[n][2]` 存储计算结果，避免重复计算：
- `memo[i][0]`：第 i 天不持有的最大利润
- `memo[i][1]`：第 i 天持有的最大利润
- 初始值为 `-1`，表示尚未计算

## 三、代码实现

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int n = prices.size();
        // 记忆化数组：memo[i][0]表示第i天不持有股票的最大利润，memo[i][1]表示持有股票的最大利润
        vector<vector<int>> memo(n, vector<int>(2, -1));
        
        // 递归lambda函数，实现记忆化搜索
        auto dfs = [&](this auto&& dfs, int i, int hold) -> int {
            // 边界条件：i < 0时的处理
            if (i < 0) {
                // 不可能持有股票，返回负无穷；不持有股票，返回0
                return hold ? INT_MIN : 0;
            }
            
            // 引用当前状态的结果，避免重复计算
            int& res = memo[i][hold];
            // 如果已经计算过，直接返回缓存的结果
            if (res != -1)
                return res;
            
            if (hold) { // 当前持有股票的情况
                // 状态转移：
                // 1. 继续持有：前一天也持有
                // 2. 买入：必须前一天是冷冻期，即i-2天不持有
                return res = max(dfs(i - 1, 1), dfs(i - 2, 0) - prices[i]);
            } else { // 当前不持有股票的情况
                // 状态转移：
                // 1. 继续不持有：前一天也不持有
                // 2. 卖出：前一天持有
                return res = max(dfs(i - 1, 0), dfs(i - 1, 1) + prices[i]);
            }
        };
        
        // 最终结果：最后一天不持有股票时的最大利润
        return dfs(n - 1, 0);
    }
};
```

## 四、时间复杂度分析

- **时间复杂度**：O(n)
  - 每个状态 `(i, hold)` 只计算一次
  - 总共有 `n × 2` 个状态，每个状态的计算时间为 O(1)

- **空间复杂度**：O(n)
  - 记忆化数组 `memo` 的大小为 `n × 2`
  - 递归调用栈的深度为 O(n)（最坏情况）

## 五、状态转移图解

以示例 `[1,2,3,0,2]` 为例：

| 天数i | 价格 | dfs(i,0)（不持有） | dfs(i,1)（持有） |
|-------|------|-------------------|-------------------|
| -1    | -    | 0                 | -∞                |
| 0     | 1    | max(0, -∞+1)=0    | max(-∞, 0-1)=-1   |
| 1     | 2    | max(0, -1+2)=1    | max(-1, 0-2)=-1   |
| 2     | 3    | max(1, -1+3)=2    | max(-1, 0-3)=-1   |
| 3     | 0    | max(2, -1+0)=2    | max(-1, 1-0)=1    |
| 4     | 2    | max(2, 1+2)=3     | max(1, 2-2)=1     |

**最终结果**：`dfs(4, 0) = 3`，与示例输出一致。

## 六、优化思路

### 1. 空间优化（动态规划）

可以使用动态规划将空间复杂度优化到 O(1)：

```cpp
int maxProfit(vector<int>& prices) {
    int n = prices.size();
    if (n == 0) return 0;
    
    int dp_prev_not_hold = 0;       // i-2 天不持有
    int dp_not_hold = 0;           // i-1 天不持有
    int dp_hold = -prices[0];       // i-1 天持有
    
    for (int i = 1; i < n; ++i) {
        int temp = dp_not_hold;
        // 不持有：继续不持有 或 卖出
        dp_not_hold = max(dp_not_hold, dp_hold + prices[i]);
        // 持有：继续持有 或 买入（i-2天不持有）
        dp_hold = max(dp_hold, dp_prev_not_hold - prices[i]);
        dp_prev_not_hold = temp;
    }
    
    return dp_not_hold;
}
```

### 2. 状态机理解

可以将问题建模为状态机，包含三个状态：
- **状态0**：不持有，且不在冷冻期
- **状态1**：持有
- **状态2**：不持有，且在冷冻期

状态转移：
- 状态0 → 状态1：买入
- 状态1 → 状态2：卖出
- 状态2 → 状态0：度过冷冻期
- 各状态内部：保持原状态
