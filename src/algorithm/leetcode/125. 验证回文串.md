# 125. 验证回文串

如果在将所有大写字符转换为小写字符、并移除所有非字母数字字符之后，短语正着读和反着读都一样。
则可以认为该短语是一个 回文串 。
字母和数字都属于字母数字字符。
给你一个字符串 s，如果它是 回文串 ，返回 true ；否则，返回 false 。

示例 1：
输入: s = "A man, a plan, a canal: Panama"
输出：true
解释："amanaplanacanalpanama" 是回文串。

示例 2：
输入：s = "race a car"
输出：false
解释："raceacar" 不是回文串。

示例 3：
输入：s = " "
输出：true
解释：在移除非字母数字字符之后，s 是一个空字符串 "" 。
由于空字符串正着反着读都一样，所以是回文串。

```cpp
class Solution {
public:
    bool isPalindrome(string s) {
        int startIndex = 0;
        int endIndex = s.size() - 1;
        while (startIndex < endIndex) {
            while (startIndex < endIndex && !isChar(s[startIndex])) {
                ++startIndex;
            }
            while (startIndex < endIndex && !isChar(s[endIndex])) {
                --endIndex;
            }
            if (startIndex == endIndex) {
                return true;
            }
            char startChar = s[startIndex];
            char endChar = s[endIndex];
            if (s[startIndex] >= 'A' && s[startIndex] <= 'Z') {
                startChar = s[startIndex] - 'A' + 'a';
            }
            if (s[endIndex] >= 'A' && s[endIndex] <= 'Z') {
                endChar = s[endIndex] - 'A' + 'a';
            }
            if (startChar != endChar) {
                return false;
            }
            ++startIndex;
            --endIndex;
        }
        return true;
    }

    bool isChar(char a) {
        if (a >= 'A' && a <= 'Z') {
            return true;
        }
        if (a >= 'a' && a <= 'z') {
            return true;
        }
        if (a >= '0' && a <= '9') {
            return true;
        }
        return false;
    }
};
```
