# 82. 删除排序链表中的重复元素 II

给定一个已排序的链表的头 head ， 删除原始链表中所有重复数字的节点，只留下不同的数字 。返回 已排序的链表 。

示例 1：
输入：head = [1,2,3,3,4,4,5]
输出：[1,2,5]

示例 2：
输入：head = [1,1,1,2,3]
输出：[2,3]

```cpp
class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        if (head == nullptr) {
            return head;
        }
        ListNode* iter = head;
        while (iter->next != nullptr && head->val == iter->next->val) {
            iter = iter->next;
        }
        if (iter == head) {
            head->next = deleteDuplicates(head->next);
            return head;
        } else {
            return deleteDuplicates(iter->next);
        }
    }
};
```
