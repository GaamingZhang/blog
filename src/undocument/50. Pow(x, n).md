# 50. Pow(x, n)

实现 pow(x, n) ，即计算 x 的整数 n 次幂函数（即，xn ）。

示例 1：
输入：x = 2.00000, n = 10
输出：1024.00000

示例 2：
输入：x = 2.10000, n = 3
输出：9.26100

示例 3：
输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25

```cpp
class Solution {
public:
    double myPow(double x, int n) {
        long long N = n;
        return N >= 0 ? quickMul(x, N) : 1.0 / quickMul(x, -N);
    }

private:
    double quickMul(double x, long long n) {
        double ans = 1.0;
        double xContribute = x;
        while (n > 0) {
            if (n % 2 == 1) {
                ans *= xContribute;
            }
            xContribute *= xContribute;
            n /= 2;
        }
        return ans;
    }
};
```
