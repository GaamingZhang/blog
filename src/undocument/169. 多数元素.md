# 169. 多数元素

给定一个大小为 n 的数组 nums ，返回其中的多数元素。多数元素是指在数组中出现
 次数 大于 ⌊ n/2 ⌋ 的元素。
你可以假设数组是非空的，并且给定的数组总是存在多数元素。

示例 1：
输入：nums = [3,2,3]
输出：3

示例 2：
输入：nums = [2,2,1,1,1,2,2]
输出：2

```cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int count = 1, target = nums[0];
        for (int i = 1; i < nums.size(); ++i) {
            if (nums[i] == target) {
                ++count;
            } else {
                --count;
            }
            if (count == 0) {
                target = nums[i];
                count = 1;
            }
        }
        return target;
    }
};
```
